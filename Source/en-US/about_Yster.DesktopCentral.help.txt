SHORT DESCRIPTION
    A PowerShell module to interact with the Desktop Central API.

LONG DESCRIPTION
    This module will allow interaction with most Desktop Central API endpoints.

    Data is returned as standard PowerShell objects.

    Some API documentation is here:
        https://www.manageengine.com/products/desktop-central/api/index.html
